Explain "repaint" and "reflow" of the webpage.

"Repaint" and "reflow" are terms used to describe the processes involved in rendering webpages. Reflow calculates the layout of elements on a webpage, determining their size, position, and arrangement based on content and styles. Repaint updates the visual appearance of elements without changing their layout. It's triggered by changes to visual properties like color, background, or visibility. Both processes are essential for displaying content accurately andvcan impact performance if not managed efficiently.

How does JavaScript on the webpage affect page load speed and performance?

JavaScript on a webpage can slow down page loading because the browser has to download and execute the script before rendering the page. If JavaScript is poorly optimized or excessive, it can block rendering, causing delays in displaying content. This affects performance, making the page feel slow to users. To improve speed, it's essential to minimize and optimize JavaScript code.

What is "minification" of the script? Why is it useful to minify your scripts?

Minification is the process of removing unnecessary characters like spaces, comments, and line breaks from code to make it smaller. This makes scripts faster to download and load on web pages, improving overall website performance.

What is a JS Web Worker and when do we need it?

JS Web Worker is like a extra worker that handles demanding tasks without interrupting the main work happening on a webpage. It's useful for keeping the webpage responsive when dealing with tasks that require a lot of processing power, like complex calculations or large data operations.
