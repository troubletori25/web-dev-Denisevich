  2. What way of including internal links into the HTML page you would consider generally preferable: absolute or relative links?

    3. When you're linking pages within your own website (or in other words, using internal links), it's usually better to use relative links. These links are like giving directions from where you are. They're easier to change if you move your pages around and they make your HTML code cleaner.

    Absolute links, on the other hand, include the full website address and are better for linking to pages on other websites. They're like giving the exact GPS coordinates. Nevertheless, they can make your code harder to manage if your website structure changes. So, from my perspective, for your own website it's better to stick with relative links for simplicity.

  3. Write about image formats used in web-development. What is the difference between JPG, GIF and PNG formats? What is WEBP image type? What is SVG?

    4. In web development, several image formats are commonly used, each with its own characteristics and suitable applications: JPEG, PNG, GIF, SVG, WebP, BMP and others.

    1. JPEG/JPG (Joint Photographic Experts Group):
      2.

      JPEG is suitable for photographs and images with many colors. It compresses files by deleting some data, leading to smaller file sizes but potential loss of quality. This format is not designed for images with transparency, but it supports millions of colors and offers adjustable compression levels.

    2. GIF (Graphics Interchange Format):
      3.

      GIF is perfect for simple animations, graphics with limited colors and images requiring transparency. It uses lossless compression for images with 256 colors or less, maintaining image quality without sacrificing file size. While GIFs support animation and transparency, they are less suitable for complex images due to their limited color palette.

    3. PNG (Portable Network Graphics):

      4. PNG is ideal for images with sharp edges, text, or transparency. It employs lossless compression, preserving image quality without compromising file size. PNG supports transparency, making it suitable for logos, icons, and graphics with transparent backgrounds. Although PNG files tend to be larger than JPEG and GIF, they maintain high quality, particularly for images with text or sharp edges.

    4. WEBP Image Type:
      5.

      WEBP is an image format developed by Google. It provides high-quality compression for images on the web, resulting in smaller file sizes compared to formats like JPEG and PNG. WEBP supports both lossy and lossless compression, as well as transparency and animation. It's commonly used for web graphics, photographs and animated images to improve website loading times and performance. However, it's essential to consider browser support and compatibility when using WEBP, as not all browsers may fully support this format.

    5. SVG (Scalable Vector Graphics):
      6.

      SVG is a vector graphics format used for displaying scalable images on the web. (The listed above are raster image formats). Unlike JPEG or PNG, SVG images are based on mathematical formulas, allowing them to be resized without losing quality. SVG supports shapes, paths, text and gradients, making it ideal for logos, icons and illustrations. It's widely supported by modern web browsers and can be manipulated using CSS and JavaScript for interactive effects.
